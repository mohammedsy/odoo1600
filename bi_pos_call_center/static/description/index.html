POS Call Center with Branch Transfer Order Odoo Apps

POS Call Center Order Transfer odoo app helps the user to operate a call center and call center branches in a point of sale sessions. Users can transfer orders to multiple call center branches from the main call center and add products, quantities, discounts, and the price will be transferred to that branch. While transferring the order from the branch sent order from one branch added as pos cart on another branch.

Features

Transfer Order to Multiple Session

User can transfer order to multiple session of different customer.

Receive Order from Multiple Session

User can receive order from different session of different user.
Demo Request Video Tutorial

POS Call Center Configuration

Under Point of Sale > Configuration > Settings menu user needs to enable the "Is Call Center" option to send an order from that session.

Under Point of Sale > Configuration > Settings menu user needs to enable the "Is a branch" option to receive the order from another pos session.

Transfer Order

If the "Is Call Center" option enables then the user can see the "Transfer Order" button on point of sale screen.

On clicking the "Transfer Order" button user can see a popup, here user can select the order date time, order branch, order priority and enter the order note.

Users can transfer and print orders by clicking the "Send and Print" button, or send the order by clicking the "Send Order" button.

User can see the printed receipt for the transferred point of sale order with the reference number.

POS Call Orders

Under Point of Sale > Orders > POS Call Orders menu user can see transferred pos orders.

User can also see the branch on point of sale orders.

Receive Order

User can receive an orders from different multiple point of sale sessions.

On clicking the "Receive Order" button user can see the order list, user can print the order receipt by clicking the "Print" button.

User can see the printed receipt for the pos order.

Transfer Order to Different Branch

User can deliver the order by clicking the "Deliver" button, User can also transfer the received pos order to another different branch, for the user need to select pos branch and click the "Transfer" button.

Updated Order

User can see the updated branch on the pos call order.
